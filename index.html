部署步骤（一次性，5分钟）
1. 打开 Google Sheets → 新建空白表格，命名为「AJD报价系统」
2. 菜单 → 扩展程序 → Apps Script
3. 删除默认代码，将「Apps Script 后端代码」全部粘贴进去
4. 保存（Ctrl+S）→ 部署 → 新建部署
5. 类型：网页应用 / 执行身份：我 / 访问权限：组织内所有人
6. 点「部署」→ 复制 URL → 粘贴到上方输入框
7. 全员下载此 HTML 文件，填入相同的 URL，即可集中管理
